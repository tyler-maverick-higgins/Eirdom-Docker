# Eirdom Docker

This repository contains the Docker stacks used throughout the Eirdom Infrastructure environment.

## Current Services

- Traefik
- Authentik
- Immich
- Open WebUI
- Qdrant
- PostgreSQL
- Redis
- Homepage
- Uptime Kuma
- Wazuh
- Portainer

## Principles

- Infrastructure as Code
- Git Version Control
- Secrets Management
- Reverse Proxy
- SSL Automation
- High Availability

## Note

Sensitive information has been removed.

Example configuration files are included where appropriate.
